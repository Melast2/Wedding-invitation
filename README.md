# Приглашение на свадьбу

Красивый одностраничный сайт с персонализированными приглашениями для каждого гостя.

## Структура файлов

- **`index.html`** — админка для создания ссылок на приглашения
- **`invitation.html`** — страница приглашения (открывается по ссылке)

## Как использовать

### Для администратора (создание приглашений):

1. **Откройте `index.html` в браузере** — откроется админка
2. **Выберите имя гостя** из выпадающего списка
3. **Скопируйте персональную ссылку** — нажмите кнопку "Копировать"
4. **Отправьте ссылку гостю** через мессенджер, email или SMS

### Для гостя:

Гость просто открывает полученную ссылку — приглашение уже будет персонализировано с его именем!

## Как добавить имена гостей

1. Откройте **`index.html`** в текстовом редакторе
2. Найдите строки с `<option value="Имя">Имя</option>` (около строки 42)
3. Добавьте новые строки по аналогии:
   ```html
   <option value="НовоеИмя">НовоеИмя</option>
   ```

## Как изменить данные о свадьбе

Все данные находятся в файле **`invitation.html`**:

- **Имена жениха и невесты** — найдите строку с `<h1 class="names">` (около строки 463)
- **Дата свадьбы** — найдите строку с `wedding-date` (около строки 468)
- **Место проведения** — найдите секцию `location-section` (около строки 541)
- **Программа дня** — найдите секцию `roadmap-section` (около строки 486)
- **Дресс-код** — найдите секцию `dresscode-section` (около строки 555)

## Пример работы

1. Откройте `index.html`
2. Выберите "Анна" из списка
3. Скопируйте ссылку: `invitation.html?guest=Анна`
4. Отправьте ссылку Анне
5. Анна откроет ссылку и увидит: "Дорогая **Анна**! Паша и Саша приглашают тебя..."
